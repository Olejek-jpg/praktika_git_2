GitHub — это крупнейший веб-сервис для хостинга IT-проектов и их совместной разработки. Если Git — это локальная программа на компьютере для отслеживания версий, то GitHub — это "облако" (социальная сеть для разработчиков), где эти репозитории хранятся, чтобы к ним можно было получить доступ с любого устройства или работать над кодом в команде.

git push нужна для того, чтобы загрузить твои локальные изменения с компьютера в удалённый репозиторий на GitHub. 
git pull нужна для обратного действия: она скачивает последние изменения из удалённого репозитория и автоматически сливает их с твоей локальной версией проекта.
